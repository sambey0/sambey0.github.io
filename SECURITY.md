# Security Policy

## Supported Versions

We currently support the following versions of the project:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing us at [security@example.com]. We will respond to your report within 72 hours and work with you to resolve the issue promptly.

Please do not disclose the vulnerability publicly until we have had a chance to address it.
